# Permissions
* permission
    * r read 
    * w write 
    * x execute 
* 3 sets of people
    * owner
    * group 
    * others 
* View Permissions  `ls -l `
* Change Permission `chmod [permissions] [path]`  
    * [ugoa] - user (or owner), group, others, all
    *  granting or revoking the permission - indicated with either a plus ( + ) or minus ( - )
    * read ( r ), write ( w ) or execute ( x )
